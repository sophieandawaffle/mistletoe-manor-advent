üîë Cipher Key

Caesar Cipher

Each letter is shifted by a fixed number of positions in the alphabet. A shift of +3 means A‚ÜíD, B‚ÜíE, C‚ÜíF, etc.

Atbash Cipher

Each letter is replaced by its mirror position: A‚ÜîZ, B‚ÜîY, C‚ÜîX, etc.

Reverse Text

The entire message is written backwards. Read from right to left to decode.

ROT13

A special Caesar cipher with a shift of 13. Since there are 26 letters, applying ROT13 twice returns the original text.

Keyword Cipher

Uses a keyword to create a substitution alphabet. Remove duplicate letters from the keyword, then append remaining alphabet letters.
